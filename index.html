🔐 SafeZone Plugin

Version: 1.0
Author: James
API: Spigot 1.13+

📦 Overview

SafeZone is a lightweight Minecraft plugin that allows server admins to create protected zones where PvP, mob damage, and block breaking are disabled. Perfect for spawn areas, shops, or safe havens.

⚙️ Commands

  • /safezone set <name> <width> <length> — Creates a zone centered on the player.
  • /safezone remove <name> — Removes the specified zone.

🧠 How It Works

  • Zones are rectangular and centered on the player’s current location.
  • Protection includes:
    • Disabling PvP
    • Preventing mob damage
    • Blocking block breaking
  • Zones are stored in memory (config persistence coming soon).

🧾 plugin.yml

name: SafeZone
main: me.james.safezone.SafeZone
version: 1.0
api-version: 1.13
commands:
  safezone:
    description: Manage safe zones
    usage: /safezone <set|remove> <name> [width] [length]

📁 Maven Setup

Use spigot-api with scope provided in your pom.xml. Java 8 recommended.
